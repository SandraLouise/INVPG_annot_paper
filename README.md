# INVPG_annot_paper

Compilation of the scripts used to produce the datasets and results showed in INVPG_annot paper.

- dir `data/` : scripts to generate simulated datasets and paths/links to data files (real and simulated)
- dir `expes/` : scripts and command lines to run the experiments



## Paper citation

Romain, S., Dubois, S., Legeai, F., & Lemaitre, C. (2025). Investigating the topological motifs of inversions in pangenome graphs. *bioRxiv*, 2025-03, https://doi.org/10.1101/2025.03.14.643331.
